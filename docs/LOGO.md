# IBIS Logo

## Logo File

Logo IBIS (Integrated Bonded Inventory System) menggunakan file:

- **network-security.png** - Logo utama yang menampilkan shield dengan padlock dan koneksi node

## Logo Description

Logo menampilkan:

- Shield (perisai) berwarna cyan/turquoise yang melambangkan keamanan dan perlindungan
- Padlock (gembok) berwarna kuning/gold di tengah yang melambangkan sistem bonded warehouse yang aman
- 6 node koneksi di sekitar shield yang melambangkan integrasi dan konektivitas sistem

## Warna Brand

- Primary: Cyan/Turquoise (#7FDBFF)
- Secondary: Navy Blue (#1E3A8A)
- Accent: Yellow/Gold (#FCD34D)

## Penggunaan Logo

### Di Login Page

Logo sudah digunakan di halaman login dengan ukuran 100x100px

### Di Komponen Lain

```html
<img src="assets/images/network-security.png" alt="IBIS Logo" />
```

### Update Favicon (Opsional)

Untuk mengganti favicon browser, convert network-security.png ke format .ico dan ganti file `public/favicon.ico`
